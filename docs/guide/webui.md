# WebUI 面板

Qtine 内置一个现代化的 Web 管理面板，采用 Material Design 3 风格，无需登录服务器即可完成所有管理操作。

## 访问方式

启动 Qtine 后，在浏览器访问：

```
http://localhost:4990/webui
```

首次启动时，命令行会打印 Token，复制粘贴到登录框即可。

::: tip
Token 也会保存到 `data/token.txt`，忘记时可以随时查看。也可以在 WebUI 设置页显示/复制 Token。
:::

## 功能模块

### 仪表盘

- 运行状态、版本号、运行时长
- 消息收发统计
- 适配器连接状态
- 插件数量统计

### 插件管理

- 查看所有插件（内置 + 外部）
- 启用 / 禁用插件
- 重载插件配置
- 通过 .zip 包导入新插件
- 删除外部插件

### 适配器管理

- 查看适配器状态（连接中 / 已连接 / 已断开）
- 查看连接的 Bot 账号
- 重连适配器
- 通过 .zip 包导入新适配器

### 日志查看

- 实时滚动显示日志
- 按级别过滤（DEBUG / INFO / WARNING / ERROR）
- 内存缓冲最近 500 条日志
- 自动刷新

### 系统设置

- Token 显示 / 隐藏 / 复制
- 修改 WebUI 配置
- 查看系统信息

## 远程访问

如果 Qtine 部署在服务器上，可以通过以下方式远程访问 WebUI：

1. **直接访问**：`http://服务器IP:4990/webui`
2. **反向代理**：通过 Nginx 反代，参考 [反向代理](/guide/reverse-proxy)

::: warning
生产环境建议启用 Token 鉴权并配置反向代理，避免直接暴露端口。
:::
