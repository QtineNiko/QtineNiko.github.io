# Qtine 是什么

Qtine 是一个基于 Python + Flask + WebSocket 的**模块化聊天机器人框架**，灵感源自 AstrBot。它采用插件化架构，支持 OneBot V11 协议，可以无缝对接 NapCat / LLOneBot 等 QQ 协议端，让你专注于业务逻辑，快速搭建自己的 QQ 机器人。

## 核心特性

### 🧩 插件化架构

Qtine 采用清晰的插件系统：

- **热加载**：运行时动态加载、卸载插件
- **多类型匹配**：命令、正则、关键词三种消息触发方式
- **隔离性**：每个插件独立运行，互不影响
- **一键导入**：通过 WebUI 上传 .zip 包导入第三方插件

### 🔌 多平台适配器

目前内置 OneBot V11 适配器，支持：

- **反向 WebSocket**：NapCat 主动连接 Qtine（推荐）
- **正向 WebSocket**：Qtine 主动连接 NapCat
- **HTTP API**：标准 OneBot V11 REST 接口

未来将支持 Discord、Telegram 等更多平台。

### 🛠️ 消息处理管道

消息从适配器进入后，会经过三阶段中间件链：

```
PRE（预处理）  →  HANDLER（命令处理）  →  POST（后处理）
```

每个阶段都可以注册多个中间件，实现黑名单过滤、频率限制、日志记录、复读检测等。

### 🖥️ WebUI 管理面板

无需登录服务器，在浏览器里就能完成所有管理：

- 仪表盘：实时查看运行状态、消息统计
- 插件管理：启用/禁用/重载/导入
- 适配器管理：查看状态、重连
- 日志查看：实时滚动，分级过滤
- 系统设置：Token 管理、配置修改

## 设计理念

Qtine 的设计初衷是**简单、灵活、可扩展**：

1. **开箱即用**：默认配置即可运行，内置 6 个常用插件
2. **聊天即管理**：常用的管理操作都能在 QQ 里完成
3. **渐进式**：从简单命令插件到复杂业务逻辑都能胜任
4. **可观测**：完善的日志和状态查询，随时知道机器人在做什么

## 适用场景

- 个人 QQ 机器人
- 群聊管理机器人
- 命令行工具集成
- 信息查询与订阅推送
- AI 对话接入（需自行开发插件）
